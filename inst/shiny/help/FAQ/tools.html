Which external tools are required?

openPrimeR requires external programs for some functionalities. To use the full potential of the tool, please ensure that the following programs are installed on your system:

MELTING
The ViennaRNA Package
OligoArrayAux
MAFFT

For a full list of suggested external programs, please click on the button in the top-right screen corner. For information regarding openPrimeR's other dependencies, we refer to R package's DESCRIPTION file.
